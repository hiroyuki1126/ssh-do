# ssh-do

## ツールのインストール方法

* 以下を実行する。

```
$ git clone https://github.com/hiroyuki1126/ssh-do
```

## ツールの実行方法

* 以下を実行する。

```
$ ./ssh-do.sh <ファイルパス1> <ファイルパス2>
```

| 引数 | 説明 |
| ----- | ----- |
| `<ファイルパス1>` | SSHでのログインに必要な情報（`<ホスト名またはIPアドレス> <ユーザ名> <パスワード>`）のリストが記載されたファイル。 |
| `<ファイルパス2>` | リモートホストで実行させるスクリプトファイル。 |
